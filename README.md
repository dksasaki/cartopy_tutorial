# Cartopy Tutorial

This tutorial covers everything you need to start making maps in Python, from setting up your terminal to plotting real data.

Start with `bash_tutorial.md` to set up your environment, then move on to `cartopy_tutorial.md` to start making maps.

## Contents

- [bash_tutorial.md](bash_tutorial.md) — opening a terminal, basic commands, and installing the Python environment with Pixi
- [cartopy_tutorial.md](cartopy_tutorial.md) — making maps with Cartopy, from a first plot to loading real data
- `pixi.toml` — the project configuration file used to install all dependencies
